# 🚀 Dev Portfolio — Front-End Development with Microsoft Copilot

A fully responsive developer portfolio website built as part of a 
5-part Microsoft Copilot project series on Coursera.

## 🌐 Live Demo
[View Portfolio](https://hack2ai.github.io/portfolio-project)

## 📋 About
This project demonstrates front-end development skills using 
HTML, CSS, and JavaScript — assisted by Microsoft Copilot AI.

## ✨ Features
- Responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Project filter by category (HTML / CSS / JS)
- Lightbox image preview
- Animated skill bars
- Contact form with real-time validation
- Accessible (WCAG compliant) & SEO optimized

## 🛠️ Built With
- HTML5 (Semantic markup)
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Microsoft Copilot (AI assistance)
